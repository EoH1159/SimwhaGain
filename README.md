# 2D SRPG 인벤토리 & 전투 데모
--
Unity 2D로 만든 간단한 **SRPG 프로토타입**입니다.  
격자 맵 위에서 유닛을 이동시키고, 인벤토리에서 무기를 선택해 공격하는 기본 시스템을 구현했습니다.
...본래는 Unity 인벤토리 구현하기였습니다.

## 개발 환경

- Unity 2D (2021 LTS 이상)
- C#

## 주요 기능

- **격자(Grid) 기반 맵**
  - 타일 자동 생성
  - 이동/공격 가능 타일 하이라이트

- **유닛 & 턴 시스템**
  - 플레이어 턴 / 적 턴
  - 이동 1회, 공격 1회 제한
  - 적 AI: 플레이어에게 한 칸씩 접근

- **인벤토리 & 무기**
  - 무기별 공격력, 내구도, 사거리(최소~최대) 설정
  - 인벤토리에서 무기 선택 후 공격
  - 내구도 0이 되면 무기 파괴

- **성장 시스템**
  - 공격 시 / 처치 시 EXP 획득
  - 레벨업 시 HP·공격·방어·행운 증가

## 조작법

- **마우스 좌클릭**: 유닛 선택, 타일 선택, 적 선택
- **커맨드 버튼**
  - Move : 이동
  - Inventory : 인벤토리 열기
  - Attack : 무기 선택 후 공격
  - End Turn : 턴 종료
- **I 키** : 인벤토리 창 열기/닫기
